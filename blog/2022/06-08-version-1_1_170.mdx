Dieses Update beinhaltet neue Funktionen und Verbesserungen

<!--truncate-->

## Allgemein

Die regelmäßigen Benachrichtigungen können Sie jetzt nutzerindividuell einstellen, so dass Sie diese entweder

- deaktivieren
- wöchentlich
- täglich
  oder an bestimmten Tagen verschicken können.

Die Einstellungen können im Profil -> Kalender vorgenommen werden.

## Auditmanagement

Erstellen Sie jetzt Auditberichte mit wenigen Klicks uns nutzen Sie Ihre vorhanden Daten besser.
Weitere Informationen finden Sie [hier](/docs/apps/audit-management#wie-erstelle-ich-einen-auditbericht).

## Blog, Dokumentenmanagement und Wiki

Als Kenntnisnehmer für Dokumente werden jetzt standardmäßig nur noch Nutzer der Software vorgeschlagen.
Da die Kenntnisnahme selbst ohnehin nur für Personen mit Nutzerkonto möglich ist, wird dadurch ein besserer Überblick über den Status der möglichen Kenntnisnahmen erreicht.
Diese Änderung bezieht sich auch auf die Funktion "Für alle zur Kenntnisnahme festlegen".
Sollten Sie trotzdem auch Mitarbeiter ohne Nutzerkonto hinzufügen wollen, können Sie im Dialog zum Hinzufügen von Personen den den Filter deaktivieren.
