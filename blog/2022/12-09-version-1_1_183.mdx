Großes Backend Update auf .NET 7 und verbesserte Schnittstellen zu Ihren ERP-Systemen.

<!--truncate-->

## Allgemein

Wir haben unser Backend auf die aktuellste Version des .NET Frameworks geupdated. Wenn Sie eine On-Premises Version von nutzen, aktualisieren Sie bitte das installierte Framework.
Der Download für die das aktuellste Framework steht [hier](https://dotnet.microsoft.com/en-us/download/dotnet/7.0) (Hosting Bunlde auswählen) zur Verfügung.

**On-Premise User bitte beachten** dieses Update beinhaltet eine notwendige Änderung an den **Verbindungszeichenfolge** für die Datebankverbindungen.
Sofern kein SQL Server mit einem gültigen Zertifikat vorhanden ist, muss explizit ein Encrypt=False der Verbindungszeichenfolge hinzugefügt werden.
Dies betrifft in der Regel zwei Stellen in Ihrer Konfiguration.

1. appsettings.Production.json ConnectionStrings:MultiTenantConnection
2. In der Datenbank für die Konfiguration Ihrer Instanz (multi-tenant) bitte die Spalte ConnectString anpassen.
