Zusätzlich gibt es noch einige Detailsverbesserungen.

<!--truncate-->

## Blog, Dokumentenmanagement und Wiki

- Genau wie die Sortierung in Kommentaren. Die Sortierung ist derzeit für die Schlagworte auf der Startseite der Dokumentenverwaltung verfügbar.
  Dort können Sie die Schlagworten alphabetisch sortieren.
  Sie können auch nach den am häufigsten oder am wenigsten verwendeten Schlagworten sortieren.
  ![Schlagwörter sortieren](https://caqadmin.blob.core.windows.net/public-screenshots/manual-screenshots/SortingTagsInDocumentManagement.png)

- Vermeiden Sie doppelte Dokumente! Beim Erstellen neuer Dokumente werden Sie jetzt auf Dokumente mit ähnlichem Titel hingewiesen und ersparen sich möglicherweise doppelte Arbeit.

## Allgemein

Sie können jetzt benutzerdefinierte Miniaturbilder zu Ihren Favoriten der Schnellzugriffslinks auf der Startseite hinzufügen.
Um dem Schnellzugriff ein Symbol hinzuzufügen auf Admin > Konfiguration > Startseite.
Klicken Sie dort auf ein Schnellzugriffelement.
Dort finden Sie Sie die Möglichkeit ein Miniaturbild für Ihre Favoriten hinzuzufügen.
![Startseite Miniaturbild](https://caqadmin.blob.core.windows.net/public-screenshots/manual-screenshots/Screenshot2022-07-25_startpage_favorite_thumbnail.png)

## Formulare

- Beim Löschen eines Formularfeldes werden die Daten jetzt auch aus allen gelöschten Formularen entfernt.
  Dies hat den Vorteil, dass in ausgefüllten Formularen nicht irgendwelche Fragment von alten Formulardesigns enthalten sind.
  Dieser Prozess ist irreversibel.

- Das nachträgliche Ändern des Feldtyps von Formularfeldern ist nicht mehr möglich. Damit verhindern wir, dass wir verschiedene inkompatible Werte miteinander vermischen.

## Zielmanagement

Currently it is possible to track the progress of your goals in more flexible way either as numbers or percentages.
Also an info has been added to the unit to give the user convinent way to customize the progress unit.
Inside the goal edit page, click on the edit button.
Inside the form there are two new fields on describes the unit metric whether it is percent or number.
The other field is for setting the unit information.
Please note that the default unit info value is '%' and the default unit metric is 'Percent'.
