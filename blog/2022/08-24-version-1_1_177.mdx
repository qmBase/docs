Dieses Update beinhaltet wesentliche Verbesserungen in der App Risiken & Chancen.
Zusätzlich gibt es noch einige Detailsverbesserungen.

<!--truncate-->

## Allgemein

Dateien vom Typ SVG können jetzt hochgeladen werden.
![allow SVG file upload](https://caqadmin.blob.core.windows.net/public-screenshots/manual-screenshots/SVGFiles.png)

### Custom Fields

Wir haben die Funktionen unserer Custom Fields erweitert, um das Arbeiten mit Custom Fields produktiver zu machen.

- Wenn Sie sehr viele Custom Fields mit einem Eintrag verknüpft haben, können Sie diese jetzt durchsuchen.
  ![Search input for custom fields](https://caqadmin.blob.core.windows.net/public-screenshots/manual-screenshots/AB_883_Intern_1.png)
- Der Wert von ausgefüllten Custom Fields kann jetzt einfach in die Zwischenablage kopiert werden, um diesen dann z.B. in eine E-Mail einzufügen.
  ![Copy custom field value to clipboard](https://caqadmin.blob.core.windows.net/public-screenshots/manual-screenshots/AB_883_Intern_2.png)
- Es kann nach Einträgen gesucht werden, bei denen das Custom Field mit dem gleichen Wert ausgefüllt ist.
  ![Search for value in index page](https://caqadmin.blob.core.windows.net/public-screenshots/manual-screenshots/AB_883_Intern_3.png)
  Das ist z.B. hilfreich, wenn Sie Custom Fields zur Priorisierung Ihrer Reklamationen nutzen.
  Angenommen Sie haben ein Custom Field mit dem Wert **PRIO 1** bei einer Reklamation ausgefüllt.
  Dann können Sie sich jetzt mit einem Klick alle anderen Reklamationen anzeigen lassen, die ebenfalls das Custom Field mit dem Wert **Prio 1** ausgefüllt haben.
  ![Index page filtered by value](https://caqadmin.blob.core.windows.net/public-screenshots/manual-screenshots/AB_883_Intern_4.png)

## Abwesenheitsplaner

Wir haben viele interne Verbesserungen am Abwesenheitsplaner vorgenommen. Die Bedienung ist jetzt schneller und flüssiger als vorher.

## Instandhaltung

- Die Zugriffsberechtigungen in der Instandhaltung können jetzt besser organisiert werden.
  Es ist jetzt möglich anderen Personen die Berechtigung zum Editieren eines Gegenstandes zu geben.
  Damit können Sie z.B. andere Personen damit beauftragen Prüfungen für einen Gegenstand durchzuführen obwohl diese nicht verantwortlich sein müssen.
- Wir haben das Formular zur Erfassung von Prüfungen überarbeitet. Selten genutzte Felder sind jetzt standardmäßig ausgeblendet.
  Zusätzlich gibt es beim Erfassen von mehreren Prüfungen in einem Schritt eine bessere Übersicht über die Prüfungen, die tatsächlich erfasst wurden.
  ![Prüfung hinzufügen](https://caqadmin.blob.core.windows.net/public-screenshots/manual-screenshots/20220531-addInspections.gif)
